About EnigmaGrid

EnigmaGrid is an interactive encryption simulator inspired by the WWII Enigma machine. Each 23x23 grid pattern generates a unique binary key that encrypts your message using XOR logic.

Uses:

  • Learn encryption concepts
  • Visualize dynamic binary keys
  • Experiment with XOR-based message encoding
  • Fun interactive educational tool
